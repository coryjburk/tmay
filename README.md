# David Eccles School of Business · MBA Career Coaching System
## Executive Self-Introduction Playbook (TMAY Mastery)

This repository contains the source code and documentation for the **Interactive Executive Self-Introduction Playbook**. Custom-calibrated for full-time MBA students and working professionals at the University of Utah David Eccles School of Business, this platform transforms the standard, open-ended interview icebreaker (“*Tell me about yourself*”) into a highly structured, margin-focused professional sales pitch.

---

## 🚀 Quick Start & Deployment

This application is built entirely as an independent, single-file client-side framework with **zero external server dependencies or cloud API key requirements**.

### Running Locally
1. Clone or download the repository to your workstation.
2. Double-click `index.html` (or the respective playbook filename) to launch the system directly inside any standard modern web browser (Chrome, Edge, or Safari recommended for native microphone APIs).

### Hosting via GitHub Pages (Free Cloud Deployment)
1. Push this file to a public or private GitHub repository named `tmay-playbook`.
2. Navigate to **Settings** ➔ **Pages** in the repository sidebar.
3. Under *Build and deployment*, set the Source dropdown to **Deploy from a branch**.
4. Choose your branch (typically `main` or `master`) and folder (`/root`), then click **Save**.
5. Your platform will be globally available via a secure URL (e.g., `https://<your-username>.github.io/tmay-playbook/`) within 1 to 2 minutes.

## 📊 Core Performance Metrics
To unlock an *Executive Presentation Fit* grading classification, candidates must align their spoken delivery with these integrated telemetry benchmarks:

| Performance Indicator | Target Threshold | Strategic Justification |
| :--- | :--- | :--- |
| **Target Duration** | 60 - 90 Seconds | Attention metrics and recruiter interest degrade sharply past 90 seconds. |
| **Pacing Tempo** | 130 - 150 WPM | Intentional pacing projects structural authority; rushing reveals cognitive stress. |

## 🛠️ Integrated Operational Modules

### 1. The 4-Part Narrative Blueprint Builder
A guided input studio that standardizes the core elements of high-impact self-introductions before speaking:
* **The Gratitude Hook:** Opens with posture, breathing room, and appreciation while anchoring the target position.
* **The Strong Headline:** A clear 1-to-2 sentence professional "through-line" defining what the candidate is universally known for.
* **The Highlight Reel:** Isolation of 2-3 data-driven, quantifiable business achievements ("Greatest Hits") rather than a chronological reading of resume history.
* **Future-Focused Alignment:** Direct application of the *Rule of 3* to show exactly where the candidate thrives and why the organization represents a logical career trajectory step.

### 2. Interactive Model Answer Bank
Pre-built, copy-pasteable script frameworks categorized by high-placement candidate archetypes:
* **The Career Switcher** (Optimized for operational pivots, tracking transferable skillsets like engineering or logistics to finance/strategy).
* **The Fast Tracker / LDP** (Calibrated for high-potential leadership development track positions emphasizing team turnaround metrics).
* **The Tech Product Explorer** (Designed around product management architectures, linking technical skill lines directly to business value margins).

### 3. Client-Side Speech Analytics Engine
Utilizes the Web Speech Recognition API to monitor performance locally within the client session:
* **Real-Time Telemetry:** Continuous word calculation, live tracking duration arrays, and active pacing calculation.
* **Vocal Trap Parsing:** String scanning regex loops that flag and count unconscious verbal filler occurrences dynamically.
* **Data Privacy Guardrail:** Absolute privacy protocol execution—zero audio streams leave the web page or touch a remote server.
